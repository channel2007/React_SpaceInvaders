# 前言

# 安裝方法
* STEP 1:建立專案.
  * create-react-app react_space_invaders

* STEP 2:進入目錄
  * cd react_space_invaders

* STEP 3:安裝套件.
  * npm install react-konva konva --save
  * npm install --save react-howler
 
* STEP 4:將本下載檔案解壓覆蓋掉原本的目錄

* STEP 5:執行
  * npm start

# 控制方法
* Enter、空白鍵：開始遊戲
* 左右鍵：左右移動角色
* M鍵：音樂、音效開關
